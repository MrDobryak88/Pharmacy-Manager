Pharmacy Manager
Pharmacy Manager — это приложение для управления аптекой, которое позволяет вести учёт запасов, продаж, клиентов, аналитики и настроек. Приложение разработано с использованием Python и PySide6, предоставляя удобный графический интерфейс для работы.

Основные возможности
Управление запасами: добавление, редактирование и удаление товаров, расчёт выручки.
Управление продажами: регистрация продаж с учётом товаров и клиентов.
Управление клиентами: добавление, редактирование и удаление информации о клиентах.
Аналитика: отображение статистики по продажам, выручке и популярным товарам.
Настройки: смена темы (светлая/тёмная), выбор цвета фона, размера шрифта, стиля кнопок и языка интерфейса (Русский/English).
Требования
Для работы с исходным кодом и запуска приложения из Python тебе понадобится:

Python 3.8 или выше
Виртуальное окружение (рекомендуется)
Установленные зависимости (pip install PySide6 matplotlib numpy)
Для запуска готового исполняемого файла (.exe) ничего устанавливать не нужно.
